# Student Performance Analytics Dashboard

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on student academic performance data and identify patterns related to attendance, department, gender, and overall scores.

## Dataset Overview

- Total Records: 30
- Total Features: 13
- Missing Values: 0

## Data Cleaning

- Verified dataset structure
- Checked data types
- Confirmed absence of missing values
- Created Average Score feature

## Department-wise Analysis

Computer Science students achieved the highest average score and attendance percentage.

Civil department recorded the lowest average score.

## Gender-wise Analysis

Female students achieved higher average scores compared to male students.

## Top Performers

Top performers were identified using a score threshold of 85%.

## Score Distribution

Most student scores were concentrated between 65 and 90.

## Outlier Detection

Outliers were identified using the IQR method and visualized using boxplots.

## Correlation Analysis

Attendance showed a positive relationship with academic performance.

## Conclusion

The analysis demonstrates that attendance and department affiliation influence academic performance. Computer Science students consistently performed better than other departments.
